# 16-bit ALU in Verilog

## Overview
This project implements a 16-bit Arithmetic Logic Unit (ALU) using Verilog HDL.

## Operations Supported
- Addition
- Subtraction
- AND
- OR
- XOR
- NOT
- Left Shift
- Right Shift

## Modules Used
- Half Adder
- Full Adder
- Half Subtractor
- Full Subtractor
- 8x1 Multiplexer

## Tools Used
- Verilog HDL
- Vivado
