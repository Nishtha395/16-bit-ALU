# 16-bit ALU in Verilog

## Overview
This project implements a 16-bit Arithmetic Logic Unit (ALU) using Verilog HDL.

## Modules Used
- Half Adder
- Full Adder
- Half Subtractor
- Full Subtractor
- 8x1 Multiplexer

## File Structure

ALU_16.v              - Top module
16Bit_fulladder.v     - 16-bit adder
fullsubtract_16bit.v  - 16-bit subtractor
full_adder.v          - Full adder
half_adder.v          - Half adder
full_subtractor.v     - Full subtractor
half_subtractor.v     - Half subtractor
mux_8x1.v             - Multiplexer

## Tools Used
- Verilog HDL
- Vivado
